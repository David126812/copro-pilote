## Résumé

Architecture d'une PWA mobile-first (React + Supabase) pour la gestion de dossiers de copropriété par les membres du conseil syndical. Auth par numéro + mdp (lien WhatsApp), isolation par copropriété (RLS), pipeline IA partagé (Claude Sonnet), notifications WhatsApp + email (Resend). 44 FRs couvertes, validée prête pour l'implémentation.
